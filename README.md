# NEXCOOP コーポレートサイト

次世代情報通信ビジネス協同組合（NEXCOOP）公式サイトのリニューアル版です。
教育・AI・通信・防災のアライアンスを表現した、1ページ構成のコーポレートサイト。

## ファイル構成

```
.
├─ index.html       … トップページ（1ページ構成）
├─ css/style.css    … スタイル（カラー・レスポンシブ・スクロール演出）
└─ images/          … 画像素材
```

## プレビュー方法

相対パスで CSS・画像を参照しているため、`index.html` をダブルクリックで開くとスタイルが当たりません。
このフォルダをルートに簡易サーバーを立てて確認してください。

```bash
# 例1: Node
npx serve

# 例2: Python
python -m http.server
```

ブラウザで `http://localhost:3000`（または `8000`）を開きます。

### GitHub Pages で公開する場合

リポジトリの **Settings → Pages** で、Branch を `master` / `(root)` に設定すると、
ブラウザで直接閲覧できる公開URLが発行されます。
